# E-Commerce SQL Project

## Overview
This project is a **practice SQL database** designed for learning and showcasing SQL skills. It models a simplified e-commerce system with Customers, Products, Orders, and Suppliers.

## Features
- Tables with **relationships and constraints**.
- Sample data for testing queries.
- Queries for beginner, intermediate, and advanced levels.
- Views and stored procedures for advanced SQL practices.

## Database Schema
- Customers
- Suppliers
- Categories
- Products
- Orders
- OrderDetails

## How to Use
1. Create the database in **SQL Server / SQL Express**.
2. Run the scripts in the following order:
   - `create_tables.sql`
   - `insert_data.sql`
   - `stored_procedures.sql`
3. Explore queries in the `queries` folder.

## Learning Objectives
- Practice `SELECT`, `WHERE`, `JOIN`, `GROUP BY`.
- Learn to use subqueries, views, and stored procedures.
- Gain hands-on experience with business-style database modeling.
